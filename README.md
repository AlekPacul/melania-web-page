# Paroki Santa Melania Bandung

Website resmi Paroki Katolik Santa Melania Bandung.

Built with [Nuxt 3](https://nuxt.com), [@nuxt/ui](https://ui.nuxt.com), and [Supabase](https://supabase.com).

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## CI/CD

- **CI**: GitHub Actions — type check, lint, build on every push
- **CD**: Vercel — auto-deploy on push to `main`
